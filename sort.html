BubbleSort

So funktioniert der BubbleSort

Bei BubbleSort werden immer zwei Elemente in eine "Bubble" gemacht und dann miteinander verglichen. Je nachdem werden sie getauscht oder bleiben an ihrer Position. Wenn sie getauscht werden, fängt man wieder (POsition 0)von vorne an um zu verhindern, dass ein Element vom Anfang dort stehen bleibt. Nacher wird immer das nächste Element genommen und mit dem vorher verglichen und so weiter. Wenn einmal alles durch ist wird es manchmal nochmal von vorne durchlaufen gelassen um alles zu überprüfen. Am Ende wird das Input in der richtigen Reihenfolge als Output zurück gegeben. Die Elemente werden nach dem Ascii Code verglichen und sortiert.
